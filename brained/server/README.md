# Brained Server (MERN starter)

This is a minimal Express + MongoDB server to pair with the `brained` frontend. It includes authentication example routes (register/login) and MongoDB connection.

Quick start (Windows cmd.exe):

1. Open a terminal in this folder: `c:\...\brained\server`
2. Install dependencies:

   npm install

3. Create `.env` (you can copy `.env.example`) and fill values:

   MONGO_URI, JWT_SECRET, PORT, CLIENT_URL

4. Run in development:

   npm run dev

Default server address: http://localhost:5000
